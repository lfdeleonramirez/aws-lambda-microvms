# Lab 1: Funciones en Python

## Objetivo
Implementar las funciones en `main.py` para que pasen los tests.

## Instrucciones
1. Abre `main.py`
2. Completa las funciones marcadas con TODO
3. Abre la terminal (Ctrl+`) y ejecuta: `pytest tests/`
4. Todos los tests deben pasar en verde

## Tiempo estimado: 30 minutos
